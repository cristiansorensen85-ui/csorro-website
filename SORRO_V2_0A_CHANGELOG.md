# Sorro OS 2.0A — Website + OS Theme Foundation

This build starts the full CSorro / Sorro OS 2.0 visual rebuild.

## Changed
- Added shared neon-blue / cyan glass design language across website and OS.
- Added reference-image-inspired dark grid background, glow layers and premium glass panels.
- Updated homepage shell, hero, dashboard preview, cards, buttons, navigation and creator strip styling.
- Updated Sorro OS app shell, sidebar, topbar, dashboard cards, module chips, CORE assistant and modal styling.
- Added the supplied Sorro OS mascot/reference image as a visual identity card on the website and OS dashboard.
- Fixed broken creator/logo image references by replacing missing images with existing bundled assets.
- Kept CSorro company branding separate from Sorro OS product naming.

## Important rule locked
- CSorro = company/brand.
- Sorro OS = the operating system/platform.
- No “Sorro Nexus” wording.
- Website and OS now use the same theme foundation.

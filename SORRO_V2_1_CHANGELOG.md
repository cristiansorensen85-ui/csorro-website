# Sorro v2.1 — Identity Release

Built from v2.0E as the first full Sorro OS identity release.

## What changed

- Reworked homepage into a premium Sorro OS launch page.
- Added v2.1 visual layer shared by website and OS.
- Strengthened CSorro/Sorro OS naming rules.
- Rebuilt Mission Control into a stronger operating-system dashboard.
- Added command dock, AI assistant styling, orbit cards and clearer module map.
- Improved responsive behaviour for mobile/tablet.
- Updated package metadata to 2.1.0.

## Brand rules locked

- CSorro is the company.
- Sorro OS is the platform.
- Existing CSorro/Sorro mark is preserved.
- Mascot is used as Sorro Assistant, not as a replacement logo.

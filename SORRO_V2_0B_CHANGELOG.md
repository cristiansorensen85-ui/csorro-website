# Sorro v2.0B — Homepage Product Reveal + OS Theme Match

This build continues the CSorro / Sorro OS 2.0 rebuild.

## Main changes

- Rebuilt the homepage as a premium Sorro OS product reveal.
- Matched the website more closely to the supplied blue neon / glass / mascot reference image.
- Added a large connected dashboard hero instead of a basic marketing layout.
- Added the supplied Sorro OS reference image into the design as the locked identity guide.
- Added Sorro Assistant / mascot treatment for the website and OS identity.
- Reworked homepage messaging so CSorro stays the company and Sorro OS stays the platform.
- Added module cards for Mission Control, Projects, Studio Review, AI Core, Assets and People.
- Added stronger shared visual language: glass panels, cyan glow, connected grid background, premium rounded nav and OS-style cards.

- CSorro is not removed or renamed.
- Sorro OS remains the operating system name.
- No invented names such as Sorro Nexus.
- The mascot is treated as the Sorro OS assistant and identity asset.
- Website and OS visual styles are now aligned rather than separate.

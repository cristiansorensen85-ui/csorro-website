# Build 050 - Theme Engine and Focus Mode

Adds a controlled appearance system for CSorro OS:

- Midnight theme (default)
- Daylight theme
- Ocean theme
- Slate theme
- Accent choices: amber, blue, green, purple, red
- Focus Mode with Ctrl + Shift + F
- Theme preferences stored in browser localStorage

Design rule: users can change the feel, not break the layout. Fonts, spacing, card structure and navigation remain controlled.

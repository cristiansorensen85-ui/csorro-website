# Sorro v2.0C — OS Mission Control Rebuild

This build rebuilds the Sorro OS dashboard from the v2.0B website foundation.

## What changed

- Rebuilt `/os/app/index.html` into a true Mission Control interface.
- Replaced the old widget dashboard with a connected OS canvas.
- Added a premium glass left rail matching the website design language.
- Added shared cyan/blue glow, grid background, ambient lighting and holographic brand treatment.
- Added Sorro Assistant / CORE integration using the approved mascot reference asset.
- Added command modal for CORE actions.
- Added project orbit, review queue, live activity, module cloud and today command stack.
- Kept CSorro as the company identity and Sorro OS as the product identity.
- Preserved the existing routes for projects, studio, calendar, people, files, messages and settings.

## Build focus

v2.0C focuses on the OS layout and makes the app feel like the website, not a separate product.

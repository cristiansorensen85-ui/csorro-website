# Backend Roadmap

## Stage 1 — Core Platform
Create accounts, profiles, workspaces, members, projects, roles and permissions.

## Stage 2 — Collaboration
Add tasks, chat rooms, project messages, activity feed and notifications.

## Stage 3 — Storage
Add Supabase Storage buckets for assets, whiteboard exports and public showcase images.

## Stage 4 — Billing
Add Stripe products, plan limits, coupons, referral codes and annual discounts.

## Stage 5 — Sorro AI
Connect Sorro AI to live project data so it can summarise, create tasks, find blockers and recommend next actions.

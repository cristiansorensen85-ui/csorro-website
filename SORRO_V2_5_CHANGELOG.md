# Sorro v2.5 — Production Readiness

This release checks and hardens the v2.4 foundation before moving into the next product sprint.

## Added
- Universal Sorro OS command palette available with Ctrl / Cmd + K.
- Quick actions for opening modules, preparing client updates and creating review checklists.
- v2.5 production readiness panel on Mission Control.
- File Manager readiness cards in Storage.
- Theme controls in Settings with local persistence.
- Notification Centre upgrade in Messages.
- Shared v2.5 CSS and JavaScript across OS pages.
- Website release strip introducing the v2.5 OS foundation.

## Fixed / Improved
- Updated package version to 2.5.0.
- Updated website and OS page titles from v2.4 to v2.5.
- Unified interaction pattern for quick actions and command search.
- Added consistent v2.5 cards, rows, pills and toast feedback.
- Kept CSorro as the company brand and Sorro OS as the product name.

## Next
- v2.6 should focus on real data wiring and deeper functional modules rather than another visual reset.
